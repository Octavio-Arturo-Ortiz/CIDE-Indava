Digitalización Exitosa del CIDE

El objetivo principal fue trasladar años de trabajos de investigación a una versión web pública, permitiendo un acceso universal a esta valiosa información.

La web garantiza la seguridad de los datos y busca facilitar el conocimiento, ofreciendo un recurso educativo accesible para académicos, estudiantes y el público en general; Representando así un paso significativo hacia la expansión de la influencia y el impacto positivo del CIDE.

DISEÑANDO EL SITIO WEB IDEAL

DISEÑANDO EL SITIO WEB IDEAL

Se enfrentó el reto de diseñar un sitio web simple para alojar información histórica y futuros resultados electorales de manera accesible. La necesidad primordial era convertir la información al formato web para permitir consultas continuas y la actualización de resultados electorales en el futuro.
